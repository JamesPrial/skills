# Skills Repository

This is a Claude Code skills plugin repository forked from anthropics/skills.

## Structure
- `skills/` - Individual skill directories (16 skills)
- `spec/` - Agent skills specification
- `template/` - Skill templates
- `.claude-plugin/marketplace.json` - Plugin manifest (owner: James Prial)

## Adding Skills
Register new skills in `.claude-plugin/marketplace.json` under `plugins[].skills[]`
